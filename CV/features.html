Website Features

  • Navbar is a Bootstrap navbar and uses Scrollspy to link to each section of the page
  • Background colour is a gradient from top to bottom through css
  • The 'hero image' is a static image in a bootstrap container
  • About Me - Boostrap grid of 2 equal width with a Boostap modal button
  • Education - Boostrap grid of 2 equal width
    • My Education - Bootstrap list
    • Grades - Bootstrap Accordian
    • Certificates - CSS zoom hover
  • Projects - Boostrap grid of 2 equal width
  • Work Experience - Bootstrap List
  • Hobbies - Bootstrap image carousel
  • Contact Me - Boostrap grid of 2 equal width
    • Form - Bootstrap form with validation
    • Contact details with LinkedIn link to new tab with CSS Icon
  • Footer - Bootstrap footer
